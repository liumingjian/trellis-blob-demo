## Quick Reference

**Tool:** agent-browser CLI

**Key Commands:**
- `agent-browser open <url>` - Navigate
- `agent-browser snapshot -i` - Get element refs
- `agent-browser fill @e1 "text"` - Fill input
- `agent-browser click @e1` - Click element
